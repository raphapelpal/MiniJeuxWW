Version Unity :
Projet 3D URP version 2020.3.23f1

Groupes trios : Chaque groupe doit avoir un trigramme différent

Pour créer un mini jeu :
 - Dupliquer le dossier "MG_Trig_NomMG (Duplicate to use)" dans MicroGames
 - Renommer le nouveau dossier en remplaçant "Trig" par votre trigramme de groupe et "NomMG" par le nom du mini jeu
 - Renommer la scène dans le dossier en remplaçant "Trig" et "NomMG" de la même manière
 - Vous pouvez maintenant travailler librement dans ce dossier

Pour modifier la vitesse et la difficulté :
 - Se rendre dans Assets/MacroGame/Resources
 - Modifier le ScriptableObject "Game Config"
